### Install Angular:
- Download Node.js
- open CMD (Admin)
- npm install -g @angular/cli@latest

### New Angular App
- ng new my-app-name
- cd my-app-name
- ng serve --open


### Use Bootstrap (or another css)
- npm install --save bootstrap@3
- update angular.json with .css file at styles
- ng serve
